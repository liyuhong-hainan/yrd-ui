# CREDITEASE YRD-UI
使用 React18+Hooks+TypeScript 定制组件库 
### install(安装)
~~~javascript
npm install yrd-ui --save
~~~
### use(使用)
~~~javascript
// 加载样式
import 'yrd-ui/dist/index.css'
// 引入组件
import { Button, Input, Menu, Icon } from 'yrd-ui'
~~~
